INSTRUCCIONES
- Esta es el Módulo OOT oficial de la E3T, dedicado principalmente para las clases de comunicaciones y demás necesidades. Un Módulo OOT, vine de "Out-Of-Tree" es el término usado en GNU rado para referirse a una libreria.

Para usar este módulo en GRC, en un nuevo PC

1. Limpiar la carpeta build, mejor dicho, borrarle todo internamente, para que quede vacia
2. desde la carpeta build realizar la instalación con los siguientes comandos:
cmake ..
make
sudo make install
sudo ldconfig

# desde la carpeta buid se invoca la herramienta cmake
# Lo que ocurre con los siguientes comandos es que todo lo que hemos hecho se 
# transforma al punto de integrarse al sistema de archivos de gnu radio, pero ojo
# ya gnu radio no va a buscar nuestro módulo en este sitio, pues le ha sacado una copia
# para incorporarla al sistema de archivos de gnu radio, que tiene un lugar de almacenamiento especial.
# Por lo anterior, nuestro módulo puede ser movido sin problemas a cualquier parte. 
# Por ejemplo, se puede compartir con los estudiantes o llevarlo a otro computador.
# Antes de compartir con otras personas un módulo es importante limpiar la carpeta build, 
# pues ella tiene información del último computador en que fué instalado. En resumen,
# lo que debe hacer la otra persona es: entrar con ubuntu al módulo, luego a la carpeta build y continuar
# con los siguientes comandos.
